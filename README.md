# 谷歌机器学习速成课程笔记(Notes for Google MLCC)

## 适合读者
就我个人而言，我更愿意称这门课程为：给工程师或开发者准备的机器学习课程，课程结合了理论与实践，并在两者之间有很好的权衡。
